# 구현할 기능 목록

## 기능별 구현

### 정답을 랜덤으로 뽑기
- `Random` API 사용


### 사용자에게 값 입력받기
- 입력 받은 값에 대한 Validation 구현
    - 유효성 검사 실패 시, `throw` 으로 작성
- `Console` API 사용


### 입력받은 수와 정답 비교하기
- strike, ball, 낫싱 체크하기

### 결과 출력하기
- 각 경우에 맞도록 결과 출력하기
    - strike가 없고 ball이 2인 경우
        - 2ball ... 
- 정답일 경우 재게임 요청하기
    - 1, 2 숫자를 통해 재게임 요청
